# Contribuyendo a CIBERWAR: El Ecosistema Silencioso

Gracias por tu interés en contribuir a este proyecto de inteligencia OSINT.

## 📋 Cómo Contribuir

### Reportar Errores o Información
Si encuentras información incorrecta o tienes datos actualizados sobre el ecosistema de ciberguerra:

1. Abre un **Issue** en el repositorio
2. Describe el error o la información nueva
3. Proporciona fuentes verificables (enlaces, informes, etc.)

### Añadir Nuevas Secciones
Si deseas ampliar el informe con nuevas secciones:

1. Haz un **Fork** del repositorio
2. Crea una nueva rama: `git checkout -b feature/nueva-seccion`
3. Añade tu contenido siguiendo el formato del informe
4. Envía un **Pull Request** con una descripción clara

## 📝 Guía de Estilo

- **Formato:** Markdown (.md)
- **Tono:** Profesional, basado en hechos y fuentes verificables
- **Fuentes:** Siempre citar fuentes abiertas (OSINT)
- **Clasificación:** No incluir información clasificada o no verificada

## 🔒 Seguridad

- No incluyas información personal de individuos no públicos
- No compartas datos de exploits activos o vulnerabilidades sin parchear
- Respeta la OPSEC de investigadores y fuentes

## 📬 Contacto

Para consultas sobre el proyecto, abre un Issue o contacta a través del repositorio.
